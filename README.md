# angular-browserify-demo

## Installation
###install dependencies:
<code>
npm install
</code>

## Running
###run server:
<code>
node index.js
</code>

then : http://localhost:3000/

### run build & watch:

<code>
gulp
</code>

### run unit tests:

<code>
npm run test
</code>
